# Google-Apps-Script-Smart-CRM-Sync-with-Static-Remark-Preservation

A secure Google Sheets automation that keeps dynamic and static data together —
✨ Even when dynamic CRM data changes, remarks always stay in the correct row.
✨ Features

Sync data from another spreadsheet

Maintains unique ID mapping to keep remarks attached
Protects remarks from deletion, reorder, or overwrite
Creates hidden RemarksDB automatically
Easy one-click syncing via custom menu
Ideal for CRM & lead tracking systems
